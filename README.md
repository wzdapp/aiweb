# Global Brand & Digital Expo - 品牌出海展会 + 数字化 B2B 独立站

> **项目代号**: Global Brand & Digital Expo  
> **战略定位**: 13位管理大师智慧加持  
> **开发周期**: 12个月  
> **开始日期**: 2026-03-14

---

## 📋 项目概述

### 核心功能
- ✅ 展会管理系统（线下+线上数据联动）
- ✅ 产品展示系统（多语言、3D展示）
- ✅ 客户关系管理（CRM）
- ✅ AI智能引擎（推荐、评分、客服）
- ✅ 营销自动化（邮件、短信、推送）
- ✅ 数据分析平台（ROI计算、实时看板）

---

## 🏗️ 项目结构

```
global-brand-expo/
├── frontend/           # 前端项目（Next.js）
├── backend/            # 后端项目（NestJS）
├── ai/                 # AI/ML项目（Python）
├── docs/               # 项目文档
├── database/           # 数据库设计
├── scripts/            # 脚本工具
└── tests/              # 测试文件
```

---

## 🚀 快速开始

### 前端开发
```bash
cd frontend
npm install
npm run dev
```

### 后端开发
```bash
cd backend
npm install
npm run start:dev
```

### AI开发
```bash
cd ai
pip install -r requirements.txt
python main.py
```
